# Skill Domain: Automation

## What this covers
Building no-code/low-code automation workflows (n8n, Zapier), Python scripting, GitHub Actions, and automating repetitive tasks across tools.

## Level Guide

| Level | What you can do |
|---|---|
| 1 | Can trigger and run an existing automation workflow |
| 2 | Builds basic multi-step workflows in n8n or Zapier |
| 3 | Builds complex workflows with conditionals, error handling, and API calls |
| 4 | Designs automation architecture; writes custom Python scripts; maintains CI/CD |

## Key Resources
- `scripts/` — Python scripts for this repo (pull_forms.py, etc.)
- `.github/workflows/` — GitHub Actions automations
- Tools in use: n8n, Zapier, Python, GitHub Actions
