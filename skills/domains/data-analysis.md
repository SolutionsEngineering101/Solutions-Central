# Skill Domain: Data Analysis

## What this covers
SQL queries, Power BI dashboard building, Excel/Google Sheets analysis, interpreting usage reports, and turning raw data into client-ready insights.

## Level Guide

| Level | What you can do |
|---|---|
| 1 | Comfortable with Excel formulas and basic filtering |
| 2 | Can write SELECT queries with JOINs; build simple Power BI reports |
| 3 | Writes complex SQL (CTEs, window functions); builds interactive BI dashboards |
| 4 | Designs data models; automates reporting pipelines; mentors others |

- `dashboard-data/` — JSON data files powering team dashboards
