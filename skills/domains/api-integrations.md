# Skill Domain: API Integrations

## What this covers
REST API consumption, SFTP-based data transfers, webhook configuration, third-party platform integrations (Switchfly, Darwinbox, SAP, etc.), OAuth flows, Postman/testing.

## Level Guide

| Level | What you can do |
|---|---|
| 1 | Understands what an API is; can read API documentation |
| 2 | Can test endpoints in Postman; configure a basic webhook |
| 3 | Builds and debugs integrations end-to-end; handles auth (OAuth, API keys) |
| 4 | Designs integration architecture; trains others; handles edge cases and failures |

## Key Resources
- `documents/integration-specs/` — client-specific integration specs
- `product-information/` — VC platform API docs and SFTP specs
- Internal: Switchfly integration playbook (see `playbook/entries/`)
