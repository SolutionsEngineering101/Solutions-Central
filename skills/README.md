# Skills

This folder has two purposes:

1. **`member/`** — Capability profiles for each team member. Useful for matching the right consultant to a client or solution type.
2. **`guides/`** — Reusable technical how-tos for recurring tasks (e.g. SSO setup, API integration, webhook config).

---

## Skill Levels

| Level | Label | Description |
|---|---|---|
| 1 | Awareness | Knows the concept, hasn't applied it |
| 2 | Working | Can execute with guidance |
| 3 | Proficient | Independent, handles most cases |
| 4 | Expert | Go-to person, can train others |

## Adding a Member Profile
Copy `member/_template.md` to `member/[first-name-last-name].md` and fill it in.
Commit: `skills: update [member] — [skill area]`

## Adding a Guide
Create `guides/[topic-slug].md`. No fixed template — keep it practical: context, steps, gotchas.
