# Solution Request #114 - Natixis Global Services India
**Date:** 2025-08-27  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution Request 122114::Implementation of T&Cs Acceptance for Natixis Global Services India

## Brief
The prospect currently follows a standard process for any new portal access, where employees must review and accept Terms & Conditions (T&Cs) before first-time login. They are currently managing this through Microsoft Forms. Ask: For the Vantage Circle R&R platform, the prospect wants:

Redirection flow: Employees will access the VC portal from their Workday instance.

Pre-SSO Intervention: Before completing login via SSO, first-time users should be shown a custom Terms & Conditions page.

Action required: Users must either Accept or Decline the T&Cs.

## Problem Description
Ensure compliance with the client’s internal governance and onboarding standards for any new platform, by embedding a mandatory T&Cs acceptance flow into the first-time login experience.

## Solution Given
Proposed Solution: Stateless T&Cs Display on Every Login
The T&Cs page is shown on every login, regardless of previous acceptance.
User Flow:
Employee initiates SSO login via Workday.
Redirected to Vantage Circle.
A pop-up displays the T&Cs.
The user must check “I Agree” and click “Agree & Proceed” to access the platform.
If the user clicks “Decline & Logout”, they are shown a decline page with two options:
“Go Back” to return to the T&Cs screen.
Confirm Decline, which logs the user out.
