# Solution Request #49 - Technip Energies UAE
**Date:** 2025-06-13  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Solution Request for Technip energies UAE

## Brief
Client has an R&R program that allows all employees a chance to vote for a peer colleague for a particular award nomination. Person with the highest nomination receives the award.

## Problem Description
Let's consider there are two awards. Their current reward program involves every employee casting a vote for their peer colleague against an award. The employee with the highest number of votes against a particular award wins the award. Only single vote can be given against a particular award by an employee.

We pitched profile badging in this case and also the idea of creating value based zero value awards instead of peer voting. And person receiving the highest number of times for award receives a profile badge with monetary value. However, can we have the limitation check of an employee giving only one award per quarter against an award?

## Solution Given
Here's how we can structure it:
Create Two Awards:
A Zero Value Auto Approved Award (for peer nomination)
A Quarterly Panel Award (which can be configured as a monetary award)
Nomination Configuration:
Assign the ‘Nominator’ role to all employees.
Set a quota of 1 nomination per award per quarter for each nominator under the Zero Value Award. This ensures employees can only vote once per award each quarter.
Progression Setup:
Link the Zero Value Auto Approved Award to the Quarterly Panel Award through a progression workflow.
All winners from the Zero Value Auto Approved Award will automatically progress to the Quarterly Panel Award.
Panel Review:
The Quarterly Panel Award will have two designated panelists to review the progressed nominees.
Based on the panel's review, one winner can be finalized by a designated Winner Selector.
This setup will ensure controlled voting, structured evaluation, and seamless progression from peer recognition to final award distribution, aligning well with the prospect’s requirements.
