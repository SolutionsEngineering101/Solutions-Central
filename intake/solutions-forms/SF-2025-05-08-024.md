# Solution Request #24 - Weaver
**Date:** 2025-05-08  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Clarification Required on Award Restrictions, Limits, and Notifications

## Brief
Need to confirm on some existing features and check on feasibility on some other features

## Problem Description
Award Restrictions:
Can a user be completely restricted from receiving any awards at all? I understand limits can be set, but the client is checking if a complete restriction is possible. Additionally, can we also restrict the number of points someone can receive, separate from awards?

Award Limit Visibility:
While giving an award, is there any way to check how many awards the recipient is still eligible to receive as per the set limits?

Manager Notifications:
Is it possible to trigger an automated email notification to a manager whenever their report receives a badge or award? If yes, would this still work if the post was marked as private?
Looking forward to your clarification so I can get back to the client accurately.

## Solution Given
Award Restrictions:
Can a user be completely restricted from receiving any awards at all? I understand limits can be set, but the client is checking if a complete restriction is possible. Additionally, can we also restrict the number of points someone can receive, separate from awards?
Response: We can restrict Awards by Department, Gender, Grade Group, and Entity through the Advanced Filtering option on the dashboard. If a user(or a group of users) does not qualify as per these restrictions, then they are automatically restricted completely. When such users are selected in the award workflow, no awards will be displayed since the criteria are not met.
Additionally, it is possible to restrict the total number of points a user can receive. This limit can be customized separately by country, based on a customizable time period (in days), and is applied across all users. Please note that this configuration requires testing to ensure proper implementation.
Award Limit Visibility:
While giving an award, is there any way to check how many awards the recipient is still eligible to receive as per the set limits?
Response: No, it can not be checked through the system. Once the recipient crosses the set limits, and someone tries to award them, the system will throw an error message.
Manager Notifications:
Is it possible to trigger an automated email notification to a manager whenever their report receives a badge or award? If yes, would this still work if the post was marked as private?
Response: Automated email notifications can be sent to Reporting Managers using the CC Matrix, and it would still work if the post was marked as private.
