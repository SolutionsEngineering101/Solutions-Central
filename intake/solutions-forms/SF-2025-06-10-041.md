**Date:** 2025-06-10  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution Request 4741 MPAC Top Values Widget

## Brief
Customization in feed

## Problem Description
MPAC has raised a request regarding the "My Top Values" section that currently appears in the milestone feed (bottom-left corner). As you know, these values are derived from recognitions where nominators use hashtags.

MPAC feels this placement may be misleading, as these recognitions are not directly related to milestones. They are requesting to shift the "My Top Values" section from the milestone feed to the normal (main) feed, where it contextually aligns better with recognition activities.

Could you please let me know:

If this shift is technically feasible?

If yes, what would be the ETA for implementation?

## Solution Given
We agree that displaying the Top Value widget on the Milestone page may not be the best placement. Hence, will implement a condition to ensure that if a client does not have R&R, the Top Values widget will not be shown.

@Preetam Rajkhowa, @Nibir Bora could you please look into this and implement the necessary logic? This widget appears on the Milestone and Profile pages.

@Nayan Bhuyan, we also support MPAC's request to remove this widget from the Milestone page. Since it's already available on the profile page, it might be redundant here. Please evaluate and proceed accordingly.
