# Solution Request #97 - LTTS
**Date:** 2025-08-05  |  **Status:** Open  |  **Complexity:** Medium Complexity

## Subject
Solution Request 10497 Receiver Validations LTTS

## Brief
An employee receiving multiple nominations for a same front end award should get an notification screen mentioning that the same user has been nominated by other nominators for the same award.

## Problem Description
Duplication of Nominations for the Same Individual -

While SPOT nominations are open to across LTTS, it would be beneficial to introduce a system feature that notifies users if a nomination already exists for a particular individual. A pop-up alert could inform the nominator of the existing entry, and an additional tab could be provided to submit a justification or reason for the new nomination. These reasons can later be consolidated and published collectively if the individual is selected as a winner.

## Solution Given
We’ve reviewed the requirements internally, and we have outlined the possibilities per our current product capabilities:

Limiting one award per nominee: This can be achieved through custom effort by implementing receiver validation to ensure that each user can be nominated only once for each award. 
Visibility into prior nominations: This functionality is outside the current scope of the product and may not always be meaningful — especially when nominations pertain to different projects or tasks. There’s also a potential risk of bias if previous nominations are visible.
Alternative via the upcoming ‘Boost’ feature
To address the underlying need for collaborative recognition, our upcoming ‘Boost’ feature may be a good fit.
It allows users to top up an existing award directly from the Feed.
A nominator can add their citation as 'Comments', effectively contributing their recognition without generating a separate award.
This keeps the recognition streamlined, aligns with the “one award per person” rule, and still provides space for multiple contributors to acknowledge the same effort.
We hope this approach can balance the intent behind the requirement with the capabilities and direction of the product.

I have reconfirmed point 1 with Tech. A nominee may be nominated once for each award, but cannot be nominated multiple times for the same award. For instance, if Nashreen is nominated for Award A, she cannot be nominated again for the same award by someone else. However, she remains eligible to be nominated for other awards.

Once confirmed by the client, we will need a list of all the Awards this validation should be applied to.

Total effort, including testing and grace period, will be 2 weeks, i.e., 10 mandays.
