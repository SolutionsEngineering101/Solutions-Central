# Solution Request #231 - AM Green Energy
**Date:** 2026-04-01  |  **Status:** Rejected  |  **Complexity:** Not Set

## Brief
AM Green Energy- To enable Request Budget Notification to Super Admin in the platform

## Problem Description
One of our clients, AM Green Energy, has requested that budget request notifications be enabled for the Super Admin in the portal’s bell icon. Currently, budget request notifications are sent via email to the Super Admin but are not reflected in the portal notifications.

## Solution Given
enabling actionable budget request notifications for Super Admins via the portal bell icon is not feasible due to platform security and authentication constraints, as direct redirection to the Admin dashboard would bypass required login controls. A display‑only notification was also ruled out, as it would result in a broken or misleading user experience and be treated as a product bug.
