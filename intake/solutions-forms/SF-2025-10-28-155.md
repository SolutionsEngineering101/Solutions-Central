## Solution Given
Thank you for your patience. We request that you please fill out the Solutions Form to help us avoid delays in the future.

Please find the answers inline below:

1. Tile Placement (Resource Library):
Can we move or reorder tiles on the home feed (e.g., shift the Resource Library higher up on the page)?
Answer: The tiles (Program Banner, Resource Library, Upcoming Birthday, Milestone, etc.) on the home feed appear as per predefined logic common to all users across the platform. We do not recommend frontend customizations; hence, the Resource Library cannot be moved higher.
However, we can address this requirement by hosting a link to the resources and updating the related messaging accordingly in the Program Banner image. (You can check it on the Edelman UAT instance)
Sample mockup: 

If not currently supported, is this flexibility part of any upcoming UI/UX enhancements?
Answer: While it is not currently a part of any planned enhancements @Nayan Bhuyan please evaluate if the control to re-arrange tiles/banners can be brought to the site admin.

2. Announcement Targeting:
Today, announcements can be sent company-wide or by department.
Answer: The announcements can currently be targeted using three filters: Country, Department, and City/Entity as per the client configurations.

Is there a way to target a smaller, custom audience (e.g., a defined list or specific employee group not mapped to a department)?
Answer: At present, there is no functionality for targeting announcements to custom audience lists.
 
If this isn’t yet configurable, do we have any ETA or roadmap details on more granular targeting?
Answer: We request that the client submit a Voice of Customer (VOC) through the Vantage Circle – Client Enhancement Request Form – Fill in form to help us evaluate this feature for potential inclusion in the future roadmap, based on usage frequency.
