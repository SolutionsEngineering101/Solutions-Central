# Solution Request #22 - Sinyar
**Date:** 2025-05-08  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Sinyar<>Feed Filter Requirement

## Brief
Addition of a third feed filter

## Problem Description
There is a requirement from Sinyar regarding the filter section in the platform's feed. Currently, the filter includes options for "All Dept" and "BU." Sinyar has three entities — Hospitality, Property Management, and Jaheziya — and they would like to know if these can be added as filter options or can be incorporated under BU.

In addition to the said requirement, please also confirm if any of the filters can be set as default view for the end users.

## Solution Given
As mentioned previously by Hemanga, a third filter for Entity can be implemented for the client to enable users to filter the feed to view either 'All Entities' or 'My entity'. 

Additionally, a default filter can be applied to the feed based on the entity. Employees may change this filter to suit their preferences, and their selection will be retained for future logins.
