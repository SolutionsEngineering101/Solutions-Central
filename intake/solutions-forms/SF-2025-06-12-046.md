## Subject
[JIRA] (VC-12149) Show the Deptt and Title along with Name on the search dialog

## Brief
Show the Deptt / Title along with Name on the search dialog
https://vantagecirclejira.atlassian.net/browse/VC-12149

## Problem Description
Organization have multiple people with same name. For example, Saurabh Gupta, or Ankit Sharma..

Thus, for anyone recognizing or nominating such users, just looking by name or email could be tricky.

It will be nice, if we could show some additional information such as the deptt and title along side the name..

So Like, 

Saurabh Gupta, Sr. Software Engineer, Telecom

Ankit Sharma, Sales Manager, Sales

## Solution Given
We currently show the location of a user's location by default however for certain clients we do have the department option as well.  Please note that these are the only two options that we can have.
