# Solution Request #3 - Antara Senior Care
**Date:** 2025-03-12  |  **Status:** Rejected  |  **Complexity:** Low Complexity

## Subject
Request to Add Prebuilt Message Templates for Service Yearbook   Amtara Seniopr Care

## Brief
Addition of prebuilt message templates that users can select from

## Problem Description
The client is concerned about whether their users will be able to share messages in a professional and appropriate manner for inclusion in the Service Yearbook. To address this, they have requested the addition of prebuilt message templates that users can select from.

AI-enabled feature, which allows users to generate personalized messages based on the recipient’s top qualities was suggested, still they reiterated their request for prebuilt templates as an additional option.

## Solution Given
Having pre-built message templates dilutes the entire purpose of having a personalized Service Yearbook. Hence, this request was rejected. 

However, it was highlighted that the AI message compose feature should be highlighted and the logic reworked to better assist users.
