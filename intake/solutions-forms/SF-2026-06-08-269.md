# Solution Request #269 - Bosch
**Date:** 2026-06-08  |  **Status:** Unknown  |  **Complexity:** Not Set

## Brief
Buddy Recognition for New Joiners: New joiners are assigned a buddy to support their onboarding and on-the-job learning. To encourage recognition within this program, the client would like to make a set of specific badges available exclusively to new joiners.

## Problem Description
* Only employees identified as new joiners should be able to view and award these badges.
* The badges should be used to recognize their assigned buddy partners.
* These badges must remain available to the new joiner for 5 months from their date of joining.
* After the 5-month period, the badges should no longer be visible or available for issuance by the employee.
* Other employees should not be able to view or issue these badges.
