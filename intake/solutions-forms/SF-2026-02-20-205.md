## Solution Given
Our Understanding
LTM has a Squad Award that is a team-based recognition program structured around sub-business units (currently 42 in total).
Each sub-business unit requires its own specific panelists and winner selectors.
The sub-business unit information will be provided in the Department column.
Based on previous discussions, the client would like the configurations to be completed as soon as possible.
Proposed Solution (As Per Current Architecture)
With the existing system architecture, the only feasible approach is to create separate Panel Awards for each sub-business unit.
The Department column must contain accurate sub-business information, as this is critical for creating and managing separate awards.
Sub-business-specific configurations can be managed using the Advanced Filter (Department filter).
Different panelists and winner selectors can be assigned by configuring separate awards for each sub-business unit.
