# Solution Request #152 - Opella
**Date:** 2025-10-21  |  **Status:** Rejected  |  **Complexity:** High Complexity

## Subject
Solution Request 157152 Country-specific access for HR Admin

## Brief
Request for country-specific access for HR Admin roles.

## Problem Description
We’ve received a request from the client Opella regarding enhanced access configurations. They would like the HR Admin access to be restricted based on country-specific data visibility. This means each HR Admin should be able to view and manage data only for their designated country.

## Solution Given
Currently, our platform supports department-level access controls, but there is no existing functionality to manage access at the country level via the Dashboard. We will explore the feasibility of implementing this with our Tech Team and get back to you.

We have reviewed this internally, and although the use case is valid, its implementation is quite complex and not feasible at this stage.
