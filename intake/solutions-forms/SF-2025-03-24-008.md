# Solution Request #8 - Mitsogo
**Date:** 2025-03-24  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Questions wizard combination feasibility | Mitsogo | Solutions

## Brief
Mapping multiple combinations to an award when using Question Wizard

## Problem Description
Client has used question wizard with 4 questions and 4 answer options for each question. They have identified 4 combinations for which the respective awards have been mapped. For the rest of the 252(4x4x4x4 - 4) combinations they seek to map a Spot Award 2 and Innovation Champ 2.

## Solution Given
If the combinations were limited the ideal solution would have been to map the combinations through Dashboard. Since it is not practically possible to map 252 combinations manually it was suggested by Himadri to have the message edited when no awards are mapped from 'No awards found' to 
Message for Badges- 'This selection didn’t match a badge. Explore other choices!'
Message for Award- 'This selection didn’t match an award. Explore other choices!'
BRD attached in link.
