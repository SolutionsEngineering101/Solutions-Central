# Solution Request #29 - Bosch RBIN & this can be an ideal use case for all the clients
**Date:** 2025-05-14  |  **Status:** To Product Closed  |  **Complexity:** High Complexity

## Subject
Solution_Request_3529_Bosch RBIN

## Brief
Report access to managers instead of Admin Access

## Problem Description
we currently have only Superadmin & admin access; person with admin access can go & make many admin level changes. This issue has come up at Bosch RBIn where 1 of the user activated SYB from the dashboard. --- A role will created for viewing only reports in dashboard.

## Solution Given
Our Understanding (as captured from the solution request form):

The current access hierarchy on the platform includes only Superadmin and HR Admin roles. Users with HR Admin access can make configuration-level changes on the dashboard, which may not be suitable for all users. This has led to issues like the one reported at Bosch RBIN, where an HR Admin-level user unintentionally activated the SYB workflow.

There is a clear need to introduce a new —“Report Viewer”—that provides access only to reports and dashboards without any configuration rights. This would enable managers and similar personas to access necessary data insights while ensuring platform integrity is maintained.

This is being raised as a must-have feature and is applicable not just for Bosch but as a scalable access model for all clients.
