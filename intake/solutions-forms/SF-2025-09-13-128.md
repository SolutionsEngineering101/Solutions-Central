# Solution Request #128 - Hyundai Capital Canada
**Date:** 2025-09-13  |  **Status:** To Product Closed  |  **Complexity:** Medium Complexity

## Subject
Solution Request::135128 Explore Feasibility of Adding Multiple Winner Selectors for Panel Awards

## Brief
Two winner selector in Panel award

## Problem Description
Hyundai has requested adding two winner selectors for the panel award. This seems like a valid request because if there is only one winner selector and they are on leave or unavailable, the winner cannot be announced. Having two winner selectors ensures that the announcement can proceed even if one is unavailable.

## Solution Given
A separate delegation role will be created specifically for the Winner Selector.
When delegated, the delegate will inherit the same functions as the original Winner Selector.
The delegate must act within the Winner Selection period configured during setup.
There will be no bypass rule — if the timeline is exceeded, it will not auto-shift to the delegate.
All other Winner Selector functionalities will remain unchanged.
