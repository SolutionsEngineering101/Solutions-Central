## Subject
Solution request:188185::Proposal to Make Mandatory Hashtag Option Site Admin–Driven

## Brief
Proposal to Make Mandatory Hashtag Option Site Admin–Driven

## Problem Description
Request you to check if the enablement of the mandatory hashtag option can be made site-admin driven, as this task is currently being handled by Tech Ops.
Similar to the attachment option, which was earlier dependent on backend configuration and has now been made site-admin driven. This change would allow better utilization of the feature as and when required by the client, with reduced turnaround time and faster resolution.

## Solution Given
The Mandatory Hashtag option will be added under the Additional Services section, similar to the other configurable checkboxes. This will allow site admins to enable or disable the mandatory hashtag requirement directly from the admin panel without Tech Ops involvement.We will be taking this up in an upcoming sprint.
