# Solution Request #59 - Vitesco Technologies Private Limited
**Date:** 2025-06-30  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution_Request_6559_Vitesco Technologies Private Limited

## Brief
We need to create a feature wherein Super admin's action is not considered as the final action

## Problem Description
The workflow for the client, Vitesco Technologies, is quota-based and includes multiple levels of approval. The final approval, or quota holding, is designated to the third-level approvers. However, if the Super Admin appears as a mid-level approver within the workflow, the system treats him/her as the final approver or quota holder. As a result, the Super Admin receives an error message indicating insufficient quota to approve the nomination.
