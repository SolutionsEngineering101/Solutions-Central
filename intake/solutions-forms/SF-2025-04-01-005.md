## Problem Description
Below are the queries I received from a client
 
Historic Nominations - User should be able to view historic nominations received by team members
 
Manager should be able to view -
No. of times his team members have been nominated (no of nominations against each team member)
No. of Awards and badges received by each member
Category of Awards and badges received by each team member

What types of information can be delivered through SMS, such as OTPs, recognition notifications, transactional details, and more?

## Solution Given
Answer: Any user can view colleagues' approved Nominations by visiting their profile if the 'Search' function is enabled for the organization.
 
Answer: This information can be extracted from the downloadable CSV report in Manager Insights. If they wish to see only the approved nominations, then it is directly available in Manager Insights.

Answer: Available inside Manager Insights. (Leaderboard too for other users)

Answer: Only the names of the Awards and Badges are visible in Manager Insights and User Profiles.

Answer: OTPs and transactional details can be sent through SMS if the users don't have email ids. The recognition notifications are sent via WhatsApp, not SMS.
