## Subject

## Brief
error message update

## Problem Description
Update default error message

## Solution Given
When a Nominator initiates a group or single-user nomination, the system must validate the selected award and denomination against the configured points rules. Upon clicking "Next," in addition to existing checks, a points validation is triggered. If any validation fails, the system will display a clear and specific error message, and the Nominator must resolve the issue by either removing the affected user(s) or adjusting the points. The nomination cannot proceed until all validation errors are resolved. This validation logic is a global change and applies consistently across all clients and users on the platform.
