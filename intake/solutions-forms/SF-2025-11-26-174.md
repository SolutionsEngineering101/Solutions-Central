# Solution Request #174 - Mitsubishi Chemicals America
**Date:** 2025-11-26  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution Request::177174:Customized text to be populated during the winner selector window in panel awards request from client.

## Brief
Customized text to be populated during the winner selector window in panel awards.

## Problem Description
Can we populate a message during the winner selector window so that the winner selectors are aware that the selection is a one time process.This will help them realise and utilise the window's time period to confirm the winners and select all of them at once to end the process.
 
right now, it might be very confusing for them and they might end up selecting one or two winners in a day and submitting, which will result the process to end.

## Solution Given
The requested changes to add the message in the winner selector before final submission have been completed and deployed to PROD.
The message added is:
“Winner announcement is a non-reversible submission, so please review and finalize your selection before submitting.”
