# Solution Request #63 - Omron Automation Private Limited
**Date:** 2025-07-07  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Solution Request 6963 'Award not found' message, Omron

## Brief
1. Change in error message on the platform when Manager (L1) and Manager (L2) is selected for nomination
2. Change in error message for Exceeding 6 Nominee Limit for Team Awards while nominating

## Problem Description
In both the cases the error message that gets displayed is : "No awards found". For the first case, the message should show "You cannot nominate your Manager and L2". For the second case, the message should show "You have exceeded the maximum limit of 6 nominations for a team award".

## Solution Given
Solution: Replacing the generic message “No awards found” with a specific message for each individual use case isn’t feasible. Instead, we recommend guiding users through a question wizard that helps them choose the correct award type from the start.
For example, presenting questions like

“What type of award is it?” 
Individual or
Team (Max team size - 6) 

will help users understand the rules and make valid selections.

As the next step, we request you to share all the specific use cases where tailored guidance or error messages are needed, so we can design the question wizard accordingly.
