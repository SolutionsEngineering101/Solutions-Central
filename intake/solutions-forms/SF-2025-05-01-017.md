# Solution Request #17 - BLUESTAR INDIA
**Date:** 2025-05-01  |  **Status:** To Product Closed  |  **Complexity:** Medium Complexity

## Subject
Query 23.17 Bluestar Employee Profile

## Brief
Bluestar India would like to add Business Group and Location on the user profile.

## Problem Description
In the user profile right now only name and department is visible. The client wants extra attributes to be shown for better identification like - Business Group and Location of the user. This is when the person is searched for in the "Search Bar" --- Meeting to be arranged with Product team by Ankit

## Solution Given
It will be handled by the @Product Vantagecircle Team. 

After internal discussions regarding this request, we have identified the following merits and demerits:
Demerits:
This is a UI-level change that will impact all existing clients, potentially requiring coordination and communication.
Additional API calls may be necessary, which could affect overall system performance.
Not all clients may be comfortable exposing these details; in the past, some have explicitly restricted profile visibility.
Merits:
Provides a more comprehensive view of employees, enriching the user experience.
The data could support future enhancements in recognition workflows or personalized redemption flows.
Simplifies the selection process during recognition, especially when multiple users share the same name.
