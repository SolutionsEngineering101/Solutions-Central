# Solution Request #281 - Sunpharma
**Date:** 2026-07-10  |  **Status:** Unknown  |  **Complexity:** Not Set

## Brief
The announcement emails we receive currently include a "Log In Now" button, which redirects users to the homepage instead of the specific announcement.

Is there a way to have this button redirect users directly to the Announcement page instead?

Previously, when the email did not include the full announcement content, it had a "View Announcement" button that redirected users directly to the announcement. After the change to include the entire announcement in the email, the button was replaced with "Log In Now," which redirects users to the homepage instead.

It would provide a much better user experience if the button could take users directly to the relevant announcement page.
