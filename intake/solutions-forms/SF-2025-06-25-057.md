# Solution Request #57 - Opella Healthcare Group (A Sanofi Consumer Healthcare business unit)
**Date:** 2025-06-25  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Solution Request 6559 Opella Budget Automation

## Brief
The client wants to allocate budget to all the users for all 47 unique regions.

## Problem Description
The client is set to keep all the users in the nominator role and allocate budgets for all across all 47 unique regions.As the user count is quite high i.e 11,000 this would be a cumbersome manual task to manually upload budgets for all across every region.There are no level of approvers as everyone would be a nominator.We need to find a workaround to make the budget allocation a hassle free and easier task.

## Solution Given
The request was discussed offline the same day and Budget Automation was suggested to address this. An approach note has been shared for the same.
