# Solution Request #183 - Volkswagen India Technology Solutions
**Date:** 2025-12-17  |  **Status:** Solution Given Closed  |  **Complexity:** Low Complexity

## Subject
Solution Request 186183 Volkswagen India Technology Solutions

## Brief
Few Clarifying Questions from the Prospect:

1. Peer-to-Peer Recognition Workflow
Looking to set up a peer-to-peer recognition program on VC that runs twice a year, with each cycle open for 2–3 weeks.

Employees should be able to share a total of 21 badges during each cycle.

Controls are required to prevent badge misuse, including: Limits to ensure a single recipient cannot receive multiple badges from the same sender. Enforcement of sender–receiver pairing restrictions.

At the end of each cycle, the employee receiving the highest number of recognitions should automatically be identified and rewarded with an award along with points.

Can this end-to-end workflow be configured within the system?

2. Decimal Points Support
Is it possible to upload and manage points in decimal values?

3. Mobile App Access & Authentication
Mobile app access is critical. Since Outlook/Teams login is not enabled on personal devices due to legal restrictions, can the platform support mobile number–based login with OTP authentication?

4. Points Revocation & Notifications
Can points be revoked from an employee’s wallet, and can an automated email notification be triggered to inform the employee when this action is taken?

5. Unused Points Reminders
Is it possible to trigger automated email notifications or nudges to employees reminding them about unused points?

## Problem Description
Detailed above

## Solution Given
1. Peer-to-peer recognition
Answer: The system currently supports certain controls based on calendar months or quarters. However, this specific, twice-yearly, cycle-based setup is a specialized use case. We will need additional time to validate feasibility with our Tech team and expect to provide a confirmation by early next week.

Additionally, while the system records recognition data, it does not have an automated feature to grant an award to the employee receiving the highest number of recognitions. The HR Admin can identify and reward the top recipient manually using either the dashboard reports or the leaderboard.

2. Decimal Points Support
Answer: No, the system can only handle integer values.

3. Mobile App Access & Authentication
Answer: The mobile app supports OTP-based login. By default, OTPs are sent to the employee’s official email ID. Only if an email ID is not available in the system, the OTP will be sent to the registered mobile number.

4. Points Revocation & Notifications
Answer: Yes, points can be revoked from an employee’s wallet through the HR Admin Dashboard. However, the system does not currently trigger an automated email notification when points are revoked.

5. Unused Points Reminders
Answer: Yes, the system automatically sends reminder emails to employees with unused points. These reminders are triggered based on predefined conditions, such as a minimum unused point threshold (250 in India) and inactivity in the last 30 days.
