## Subject
Solution Request : 275275 : TML Error Message

## Brief
Request for Detailed Error Messages During Monetary Award Nomination for Tata motors

## Problem Description
We have received a new requirement from TML, comp id 443.

They have observed that the system displays a generic standard error message when a nominator nominates a monetary award to an employee during the nomination stage.
 
Since there can be multiple reasons for such failures, the generic error message makes it difficult for us to identify the actual reason for the failure.
 
Request your support in providing more specific and descriptive error messages during the nomination process.
 
This would help us in accurately identifying the failures and resolving such issues more efficiently.
