# Solution Request #167 - Celebal Technologies
**Date:** 2025-11-07  |  **Status:** Open  |  **Complexity:** Low Complexity

## Subject
Request for Profile Picture Transfer from Darwinbox to Vantage via API Integration

## Brief
Request for Profile Picture Transfer from Darwinbox to Vantage via API Integration

## Problem Description
We have received a request from Celebal Technologies to enable the transfer of profile pictures from Darwinbox to Vantage via API integration or to provide an option for bulk profile picture upload through the admin dashboard.
Request you to please check the feasibility and confirm.

## Solution Given
We have received similar requests earlier and had consulted with the Tech team regarding this. Unfortunately, profile picture transfer via API is not feasible due to technical reasons mentioned below:
Our system uses Cloudinary to store images, generating a unique link for each photo upon user upload. These links are then saved in our database. This process must be initiated by the user and cannot be performed on their behalf.
Image extraction through HRMS/HRIS integration is also not possible, as those systems may store profile pictures in formats that are incompatible with our Cloudinary-based storage requirements.
