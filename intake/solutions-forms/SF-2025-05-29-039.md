# Solution Request #39 - Perceptive / dal
**Date:** 2025-05-29  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution Request 4539 Reminder Mail for Approvers

## Brief
Reminder notifications/email/alert under to-do-list for approvers to approve the pending awards.

## Problem Description
I'd like to request an enhancement related to the nomination approval process on the platform. Currently, approvers do not receive consistent reminders for pending nominations, which often leads to delays in the overall recognition flow.
To address this, it would be beneficial to implement an automated reminder email system that notifies approvers at a set frequency (e.g., every 3 or 5 days) until action is taken on the pending nominations. Ideally, the frequency and the number of reminders could be configurable based on the organization's preference. or a notification or display of red abstract in the to-do list segment. 
This feature would help ensure timely approvals and improve the overall user experience and engagement with the platform. We have also received these as a request from the clients.
Please feel free to let me know if you need any additional context or if we can discuss this further.

## Solution Given
The issue was discussed and it has been agreed to work on a common reminder to approvers via mail and whatsapp.

A reminder mail already exists. It has to be deployed for EU US (All) server.
