## Problem Description
We are filling out an RFQ for Maruti Suzuki India and they have a couple of questions regarding the platform functionality. So, we need your help with the best possible response to these 2 questions:
1. Do you offer the ability for HR Admins to create an R&R solution tile embedded within a core HR platform like SAP SuccessFactors?
2. Do you provide a recognition copilot that assists employees in crafting appreciation messages and generates AI-enhanced images to complement recognitions?

## Solution Given
We do have an AI-driven feature called Recognition Guide, which acts as an assistant while a user is writing an appreciation message. It suggests improvements based on defined recognition parameters and also helps paraphrase the message.
Additionally, we support the feasibility for users to generate AI-powered comments for recognition posts. Both these features would be rolled out by the end of October 25.
As for image generation, this is part of our roadmap — we plan to enable users to generate stickers that can be used within comments.
