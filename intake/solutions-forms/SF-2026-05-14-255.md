# Solution Request #255 - Amazon India
**Date:** 2026-05-14  |  **Status:** Unknown  |  **Complexity:** Not Set

## Brief
Amazon India is looking for a points redemption platform. Key requirements and open queries are as follows: requires a rewards portal with suppressed pricing visibility, merchandise-focused redemption, end-to-end fulfilment with no transportation hassles, order updates and CSV-based employee management and reporting dashboard for redemption.

## Problem Description
1. Pricing suppression: Portal must display only points to end users — no rupee values on the platform, on the shipment or inside the package/merchandise or invoices.
2. Merchandise focus: Gift cards and experiences to be disabled; only general merchandise (not company store) to be shown. Is it possible?
3. Delivery timelines: Confirm if estimated delivery timelines can be displayed during product selection or before the checkout, so the user knows when to expect the delivery.
4. Employee management: Clarify whether CSV upload process has to be done everytime for incremental employee additions and point allocations.
5. Notifications: Confirm availability of email/mobile push notifications for order status updates.
