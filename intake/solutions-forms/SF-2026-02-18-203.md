# Solution Request #203 - Bosch RBIN - 971
**Date:** 2026-02-18  |  **Status:** Solution Given Closed  |  **Complexity:** Medium Complexity

## Subject
Solution request 207203::Requirement for Creation of a Secondary Wallet for Targeted Employee Group

## Brief
Requirement for Creation of a Secondary Wallet for Targeted Employee Group.

## Problem Description
The client has requested the creation of a secondary wallet on the platform specifically to support employee incentivization initiatives. This wallet is intended to be provisioned only for a defined group of employees, with eligibility determined using “Department field/Group Type” as the unique identifier. The system should ensure that the secondary wallet is created exclusively for employees mapped to the specified group type, while remaining unavailable to others outside this category. This enhancement will enable the client to run targeted incentive programs in a controlled and structured manner.

## Solution Given
creating a secondary wallet for a specific department and removing surcharges for selected employee groups is not feasible due to platform limitations. Multi‑wallet functionality and surcharge settings apply at the company-wide level and cannot be restricted or customized for specific groups. No further action is possible on this requirement under the current system constraints.
