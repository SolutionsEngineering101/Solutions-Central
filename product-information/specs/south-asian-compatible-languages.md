# South Asian Compatible Languages

Languages in South Asia that the Vantage Circle (VC) app can run in, with their
locale IDs and the platform-specific resource directory naming used for iOS and
Android localization.

## Supported Languages

| Language | Locale ID | iOS `.lproj` | Android `values-` |
|---|---|---|---|
| Hindi | `hi` | `hi.lproj` | `values-hi` |
| Bengali (Bangla) | `bn` | `bn.lproj` | `values-bn` |
| Tamil | `ta` | `ta.lproj` | `values-ta` |
| Telugu | `te` | `te.lproj` | `values-te` |
| Kannada | `kn` | `kn.lproj` | `values-kn` |
| Malayalam | `ml` | `ml.lproj` | `values-ml` |
| Marathi | `mr` | `mr.lproj` | `values-mr` |
| Gujarati | `gu` | `gu.lproj` | `values-gu` |
| Punjabi | `pa` | `pa.lproj` | `values-pa` |
| Odia | `or` | `or.lproj` | `values-or` |
| Assamese | `as` | `as.lproj` | `values-as` |
| Urdu | `ur` | `ur.lproj` | `values-ur` |

## Notes

- **Locale ID** follows the ISO 639-1 two-letter language code.
- **iOS `.lproj`** is the localization bundle directory name used in the iOS app.
- **Android `values-`** is the resource qualifier directory name used in the Android app.
- 12 South Asian languages are currently supported.
